Мысли о хлебе¶

История вопроса¶

Однажды я пошёл за хлебом. И понял, что выбор — это сложно.

Виды хлеба¶

  • Белый

  • Чёрный

  • С отрубями

  • Без глютена

  • Тот, который с семечками

Алгоритм выбора¶

  1. Понюхать

  2. Нажать пальцем

  3. Посмотреть на цену

  4. Если есть скидка — взять два

Итог¶

Лучший хлеб — тот, который не закончился к ужину.

Цитата дня¶
