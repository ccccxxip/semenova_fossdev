## В проекте используется Makefile как единая точка входа

## установка зависимостей:
make install

## запуск:
make run

## проверить типы:
make typecheck

## проверить зависимостей:
make check-requirements

## все проверки:
make check


## Как работает проект

Makefile используется как единая точка входа.

Каждая команда вызывает инструменты из .venv/bin, поэтому не требуется активация окружения.

## Проверка зависимостей

Скрипт check_requirements.py проходит по всем файлам в src и проверяет, что все импорты есть в requirements.txt.

## Проверка типов

Используется mypy для проверки типов.

## Проверка стиля

Используется black для форматирования кода.
